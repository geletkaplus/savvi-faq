== Description ==

The Savvi FAQ plugin allows you to create a list of collapsible FAQs to display almost anywhere on your Wordpress site. However, to get the full benefits of this plugin you should have a Savvi AI account. When integrated with a Savvi AI Stem the plugin utilizes machine learning to display the FAQs in the most-clicked order, making sure your visitors see the most applicable FAQs first.

== Installation ==

**Installing the plugin**

1. Zip the savvi-faq directory and upload the plugin to your blog and activate
2. Follow the instructions on the plugin settings page

[More info on installing WP plugins](https://wordpress.org/support/article/managing-plugins/#installing-plugins)
